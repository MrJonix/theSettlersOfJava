### Prerequisites

- Java 21 or newer
- Maven

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/MrJonix/theSettlersOfJava.git
   cd theSettlersOfJava
    ```

2. Build the project:

   * With Maven:
    ```bash
    mvn clean install
    ```

3. Run the game:

   ```bash
   java -jar target/theSettlersOfJava.jar
   ```
